# Meta-Analysis Examples

## Example 1: Very Strong Finding - Growth Mindset Training

**Original Hypothesis:** "Teaching people growth mindset increases academic performance"

### Replication Data (23 replications submitted)
```
Successful: 21
Failed: 2
Replication Rate: 91.3%
```

### Generated Meta-Analysis Output

```
🔬 Synthesized Meta-Analysis
23 replications

📊 Replication Summary
├─ Total Replications: 23
├─ Successful: 21
├─ Replication Rate: 91.3%
└─ Strength of Evidence: Very Strong ✅

📈 Effect Size
├─ Mean Effect Size: 0.72 (Large)
└─ 95% Confidence Interval: [0.61, 0.83]
    (We're 95% confident the true effect is between 0.61-0.83)

📊 Consistency Across Replications
├─ Standard Deviation: 0.13
├─ Variance: 0.017
└─ Heterogeneity (I²): 28% → Consistent results
    (Small variation across replications)

⚠️ Conditions Where Hypothesis Fails
├─ "Fails in students with diagnosed depression/anxiety (effect reversed)"
├─ "Requires reinforcement in weeks 5-8 or effect fades"
└─ "Less effective in cultures emphasizing fixed intelligence (East Asian sample)"
```

### Scientific Interpretation

**Why this is Very Strong:**
- 91% replication rate (way above 80% threshold)
- 0.72 effect size (large effect, practically meaningful)
- Low heterogeneity (consistent across studies)
- Failure conditions identify boundary cases, not fatal flaws

**Real-world impact:** This is ready for deployment in schools. Know the specific populations where caution is needed (depression, weeks 5-8 reinforcement).

## Example 2: Weak/Contested Finding - Cold Water Immersion Recovery

**Original Hypothesis:** "Cold water immersion speeds athletic recovery"

### Replication Data (27 replications submitted)
```
Successful: 14
Failed: 13
Replication Rate: 51.9%
```

### Generated Meta-Analysis Output

```
🔬 Synthesized Meta-Analysis
27 replications

📊 Replication Summary
├─ Total Replications: 27
├─ Successful: 14
├─ Replication Rate: 51.9%
└─ Strength of Evidence: Weak ⚠️

📈 Effect Size
├─ Mean Effect Size: 0.38 (Small)
└─ 95% Confidence Interval: [0.15, 0.61]
    (Confidence interval spans small-to-medium effect)

📊 Consistency Across Replications
├─ Standard Deviation: 0.31
├─ Variance: 0.096
└─ Heterogeneity (I²): 82% → High variation
    (Results vary substantially across studies)

⚠️ Conditions Where Hypothesis Fails
├─ "No effect in eccentric exercises; only concentric movements show benefit"
├─ "Effect reverses in cold climates (worsens next-day soreness)"
├─ "Effect depends on psychological belief (placebo component)"
├─ "Fails in female athletes; significant sex-based difference"
├─ "Only works within 30min of exercise; delayed immersion shows no benefit"
```

### Scientific Interpretation

**Why this is Weak:**
- Only 52% replication rate (coin flip, basically)
- 0.38 effect size (small, if it exists)
- **82% heterogeneity** (huge variation in results!)
- Many failure conditions suggest complex moderators

**What the data tells you:** 
- This is a **real phenomenon** in some contexts
- But **highly context-dependent**
- Sex differences, timing, exercise type all matter
- Possibly large placebo component

**Research recommendation:** Don't use as standard protocol. But high-quality research opportunity to map out the conditions properly.

## Example 3: Moderate Finding with High Heterogeneity - Social Media Detox

**Original Hypothesis:** "Taking a 1-week break from social media improves mental health"

### Replication Data (31 replications submitted)
```
Successful: 24
Failed: 7
Replication Rate: 77.4%
```

### Generated Meta-Analysis Output

```
🔬 Synthesized Meta-Analysis
31 replications

📊 Replication Summary
├─ Total Replications: 31
├─ Successful: 24
├─ Replication Rate: 77.4%
└─ Strength of Evidence: Moderate 🟡

📈 Effect Size
├─ Mean Effect Size: 0.58 (Medium)
└─ 95% Confidence Interval: [0.40, 0.76]

📊 Consistency Across Replications
├─ Standard Deviation: 0.28
├─ Variance: 0.078
└─ Heterogeneity (I²): 71% → High variation
    (Moderate-to-high variation across studies)

⚠️ Conditions Where Hypothesis Fails
├─ "Effect doubles for heavy users (>5hrs/day) vs light users (<1hr)"
├─ "No effect in FOMO-anxious individuals; anxiety increases when access restored"
├─ "Paradoxical increase in anxiety during first 3 days for some"
├─ "Does not improve deep work in people who use socials for networking"
├─ "Effect stronger for Instagram/TikTok; minimal for LinkedIn/professional networks"
├─ "Requires user agency (voluntary detox works; forced detox shows no benefit)"
```

### Scientific Interpretation

**Why this is Moderate:**
- 77% replication rate (above threshold but not robust)
- 0.58 effect size (medium, but variable)
- **71% heterogeneity** = strong boundary conditions

**What the data tells you:**
- Effect is **real but highly moderated**
- Works better for:
  - Heavy users
  - Social platforms (not professional)
  - Voluntary participation
  - Users without extreme FOMO
- IMPORTANT NUANCE: First 3 days can worsen anxiety (withdrawal pattern)

**Practical application:** 
- ✅ For heavy social media users wanting to reduce: likely beneficial
- ❌ For FOMO-anxious people: could backfire
- ⚠️ Requires professional support during first week
- ❌ Won't help professionals using for networking

## Example 4: Very Weak Finding - Homeopathic Remedies

**Original Hypothesis:** "Homeopathic dilutions beyond 12C have biological effects"

### Replication Data (18 replications submitted)
```
Successful: 2
Failed: 16
Replication Rate: 11.1%
```

### Generated Meta-Analysis Output

```
🔬 Synthesized Meta-Analysis
18 replications

📊 Replication Summary
├─ Total Replications: 18
├─ Successful: 2
├─ Replication Rate: 11.1%
└─ Strength of Evidence: Very Weak ❌

📈 Effect Size
├─ Mean Effect Size: 0.08 (Negligible)
└─ 95% Confidence Interval: [-0.12, 0.28]
    (Confidence interval includes 0 = possibly no real effect)

📊 Consistency Across Replications
├─ Standard Deviation: 0.21
├─ Variance: 0.044
└─ Heterogeneity (I²): 12% → Consistent results
    (Actually consistent — consistently no effect!)

⚠️ Conditions Where Hypothesis Fails
├─ "All dilutions beyond 12C failed to outperform placebo"
├─ "Even 'successes' showed effect size indistinguishable from placebo"
└─ "No difference when blinded vs unblinded protocols"
```

### Scientific Interpretation

**Why this is Very Weak:**
- Only 11% replication rate (essentially never replicates)
- **0.08 effect size** (negligible, within noise)
- Confidence interval crosses zero = no reliable effect
- Results are **consistent in their negativity** (I² = 12% = actually strong consistency around "no effect")

**What the data tells you:**
- This is **not a real effect**
- The 2 "successful" replications likely had methodological flaws
- Placebo entirely explains outcomes
- No boundary conditions — fails universally

**Scientific recommendation:** 
- Don't pursue further (replicate-then-shift principle)
- Future research should focus on mechanism of placebo (which IS real)
- Platform has definitively answered this question

## Key Insights for Researchers

### Pattern 1: Low Heterogeneity Can Mean Different Things

**Mindset (28% heterogeneity):** Consistent effect across contexts = robust finding
**Homeopathy (12% heterogeneity):** Consistently no effect = well-established negative result

**Lesson:** High heterogeneity ≠ bad. It's good if it reveals boundary conditions!

### Pattern 2: High Replication Rate + Low Effect = Apply Carefully

If something replicates 90% of the time but with tiny effect size, the finding is:
- ✅ Real (strong replication)
- ⚠️ But practically weak (might not matter in real applications)
- 📋 Example: Small medication side effect that's rare but consistent

### Pattern 3: Moderate Rate + High Heterogeneity = Map the Boundaries

This is the **most interesting** pattern scientifically:
- Not false (too many successes)
- Not trivial (meaningful effect size)
- But complex (boundary conditions matter)
- **Next step:** Identify what causes the heterogeneity

### Pattern 4: Low Rate + Consistency = Likely False

If something fails 90% of the time with consistent methodology, it's probably not real. Don't waste time chasing boundary conditions.

## Using Meta-Analysis for Decision-Making

### For Clinicians
"Should I recommend this treatment?"
- Very Strong → Yes, with noted exceptions
- Strong → Generally yes
- Moderate → Case-by-case (depends on patient characteristics)
- Weak → No
- Very Weak → No

### For Researchers
"Should I research this further?"
- Very Strong → No (solved), apply findings
- Strong → No (solved), apply findings
- Moderate → **Yes** (map the boundaries)
- Weak → **Yes** (resolve the contradiction)
- Very Weak → No (well-falsified, move on)

### For Science Policy
"Should this be funded?"
- Very Strong → No (proven), shift to applications
- Strong → No (proven), shift to applications
- Moderate → **High Priority** (timely question, needs resolution)
- Weak → **High Priority** (resolve conflicting evidence)
- Very Weak → No (effectively disproven)

## The Value Proposition

Without meta-analysis:
- 18 cold water papers → Conflicting recommendations → Reader confused
- 31 social media papers → No clear guidance → Everyone does their own thing
- 23 mindset papers → Scattered support → Debate continues

With meta-analysis:
- Clear summary of strength
- Exact boundary conditions
- Confidence intervals for effect magnitude
- Transparent decision-making framework
- Researchers can build instead of debate

**That's the power of synthesis.**
